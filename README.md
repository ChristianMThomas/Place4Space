Place4Space

Place4Space is a simple but production-grade cloud storage web application that allows users to securely upload, manage, and view files. It demonstrates a modern full-stack architecture using Next.js, Spring Boot, PostgreSQL, Docker, and AWS.

The project is designed as a realistic example of how scalable file-storage systems are built in the real world.
